Parameters

Name

Description

view

Trust lines and balances

deliver

Amount to deliver to the dst account

src

Account providing input funds for the payment

dst

Account receiving the payment

paths

Set of paths to explore for liquidity

defaultPaths

Include defaultPaths in the path set

partialPayment

If the payment cannot deliver the entire requested amount, deliver as much as possible, given the constraints

ownerPaysTransferFee

If true then owner, not sender, pays fee

offerCrossing

If true then flow is executing offer crossing, not payments

limitQuality

Do not use liquidity below this quality threshold

sendMax

Do not spend more than this amount

j

Journal to write journal messages to

flowDebugInfo

If non-null a pointer to FlowDebugInfo for debugging

Actual amount in and out, and the result code
