StepImp::redeems

Inherited from Step. If this step is a DirectStepI and the src redeems to the dst, return true, otherwise return false.

Synopsis 
virtual
bool
redeems(
    ReadView const& sb,
    bool fwd) const;
Description

If this step is a BookStep, return false if the owner pays the transfer fee, otherwise return true.

Parameters

Name

Description

sb

view with the strand's state of balances and offers

fwd

false -> called from |9|rev(); true -> called from |9|fwd().
